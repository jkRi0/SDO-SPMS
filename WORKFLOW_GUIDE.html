How to Enable "Proceed to Landbank" Button

Complete the entire workflow to unlock the Landbank button in the Cashier module

Procurement Step 1: Create & Approve Transaction

Go to Procurement dashboard → Click "New Transaction" → Fill in:

  • Program Title: Required
  • Proponent: Required
  • Supplier: Required
  • Date (Coverage): Required
  • PO Amount: Required

After creation, the status should be: FOR SUPPLY REVIEW

Supply Step 2: Verify & Approve

Go to Supply dashboard → Find the transaction → Click "View/Update"

Set the following fields:

  • Supply Status: Set to any value EXCEPT "PENDING" or "FOR CORRECTION" (e.g., "APPROVED FOR BUDGET")
  • Supply Remarks: Optional
  • Supply Date: Set the date

Click "Save Changes"

Accounting (Pre-Budget) Step 3: Pre-Budget Approval

Go to Accounting dashboard → Find the transaction → Click "View/Update"

Set the following fields:

  • Accounting (Pre-Budget) Status: Set to any value EXCEPT "PENDING" or "FOR CORRECTION" (e.g., "APPROVED")
  • Accounting (Pre-Budget) Remarks: Optional
  • Accounting (Pre-Budget) Date: Set the date

Click "Save Changes"

Budget Step 4: Prepare DV & Fill Budget Details

Go to Budget dashboard → Find the transaction → Click "View/Update"

IMPORTANT - Fill these fields:

  • Budget Status: Set to any value EXCEPT "PENDING" or "FOR CORRECTION" (e.g., "DV APPROVED")
  • Budget DV Number: REQUIRED - Enter a DV number (e.g., "DV-2026-001")
  • Budget DV Date: REQUIRED - Set the date
  • Budget Remarks: Optional

Click "Save Changes"

Accounting (Post-Budget) Step 5: Post-Budget Approval

Go to Accounting dashboard → Find the transaction → Click "View/Update"

Set the following fields:

  • Accounting (Post-Budget) Status: Set to "FOR PAYMENT" or "FOR CASHIER – PAYMENT PROCESSING"
  • Accounting (Post-Budget) Remarks: Optional
  • Accounting (Post-Budget) Date: Set the date

Click "Save Changes"

Cashier Step 6: Process Payment & Proceed to Landbank

Go to Cashier dashboard → Find the transaction → Click "View/Update"

If all steps above are complete, you should see:

✓ All requirements from Procurement, Supply, Accounting, and Budget are complete.

Fill in (optional but recommended):

  • Cashier Status
  • Cashier OR Number
  • Cashier OR Date
  • Cashier Landbank Ref (if applicable)
  • Cashier Payment Date
Troubleshooting

Button still shows "Cannot proceed yet"?

Check the following:

  1. ALL statuses must be filled (not empty/null)
  2. NO status should be "PENDING" or "FOR CORRECTION"
  3. Budget DV Number and DV Date MUST be filled
  4. Accounting (Post-Budget) Status must be "FOR PAYMENT" or "FOR CASHIER – PAYMENT PROCESSING"

Use this tool to check: Check Transaction Status
